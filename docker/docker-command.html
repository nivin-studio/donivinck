# Docker命令

# 镜像仓库操作

# docker login

#登录docker镜像仓库
语法：docker login [OPTIONS] [SERVER]
实例：docker login -u 用户名 -p 密码

# docker search

#在镜像仓库中搜索镜像
语法：docker search [OPTIONS] TERM
实例：docker search redis

# docker pull

#从镜像仓库中拉取或者更新指定镜像
语法：docker pull [OPTIONS] NAME[:TAG|@DIGEST]
实例：docker pull redis

# docker push

#将本地的镜像上传到镜像仓库
语法：docker push [OPTIONS] NAME[:TAG]
实例：docker push myRedis:v1

# 本地镜像管理

# docker images

#镜像列表
语法：docker images [OPTIONS] [REPOSITORY[:TAG]]
实例：docker images

# docker rmi

#删除一个或多个镜像
语法：docker rmi [OPTIONS] IMAGE [IMAGE...]
实例：docker rmi redis

# docker tag

#给镜像创建新的标记
语法：docker tag SOURCE_IMAGE[:TAG] TARGET_IMAGE[:TAG]
实例：docker tag redis:4.0 nivin/redis:v4

# docker build

#通过Dockerfile文件构建一个镜像
语法：docker build [OPTIONS] PATH | URL | -
实例：docker build -t nivin/redis:v4 .

# docker history

#显示镜像的创建过程
语法：docker history [OPTIONS] IMAGE
实例：docker history nivin/redis:v4

# docker save

#将指定镜像保存成 tar 归档文件
语法：docker save [OPTIONS] IMAGE [IMAGE...]
实例：docker save -o redis_v4.tar nivin/redis:v4

# docker import

#从归档文件中创建镜像
语法：docker import [OPTIONS] file|URL|- [REPOSITORY[:TAG]]
实例：docker import redis_v4.tar nivin/redis:v4

# 容器生命周期

# docker run

#创建一个新的容器并运行一个命令
语法：docker run [OPTIONS] IMAGE [COMMAND] [ARG...]
实例：docker run --name redis_container -p 80:80 -v /data:/data -d redis:4.0

# docker start

#启动一个或多个已经被停止的容器
语法：docker start [OPTIONS] CONTAINER [CONTAINER...]
实例：docker start redis_container

# docker stop

#停止一个或多个正在运行的容器
语法：docker stop [OPTIONS] CONTAINER [CONTAINER...]
实例：docker stop redis_container

# docker restart

#重新启动一个或多个容器
语法：docker restart [OPTIONS] CONTAINER [CONTAINER...]
实例：docker restart redis_container

# docker kill

#杀死一个或多个正在运行的容器
语法：docker kill [OPTIONS] CONTAINER [CONTAINER...]
实例：docker kill redis_container

# docker rm

#删除一个或多个容器
语法：docker rm [OPTIONS] CONTAINER [CONTAINER...]
实例：docker rm redis_container

# docker pause

#暂停一个或多个容器中的所有进程
语法：docker pause CONTAINER [CONTAINER...]
实例：docker pause redis_container

# docker unpause

#恢复一个或多个容器中的所有进程
语法：docker unpause CONTAINER [CONTAINER...]
实例：docker unpause redis_container

# docker create

#创建容器
语法：docker create [OPTIONS] IMAGE [COMMAND] [ARG...]
实例：docker create -t -i redis_container bash

# docker exec

#在运行的容器中执行命令
语法：docker exec [OPTIONS] CONTAINER COMMAND [ARG...]
实例：docker exec -it redis_container bash

# 容器运维操作

# docker ps

#容器列表
语法：docker ps [OPTIONS]
实例：docker ps -a

# docker inspect

#获取容器/镜像的元数据
语法：docker inspect [OPTIONS] NAME|ID [NAME|ID...]
实例：docker inspect redis_container

# docker top

#显示容器的运行进程
语法：docker top CONTAINER [ps OPTIONS]
实例：docker top redis_container

# docker attach

#连接到正在运行中的容器
语法：docker attach [OPTIONS] CONTAINER
实例：docker attach redis_container

# docker events

#从服务器获取实时事件
语法：docker events [OPTIONS]
实例：docker events

# docker logs

#获取容器的日志
语法：docker logs [OPTIONS] CONTAINER
实例：docker logs -f redis_container

# docker wait

#阻塞运行直到容器停止，然后打印出它的退出代码
语法：docker wait CONTAINER [CONTAINER...]
实例：docker wait redis_container

# docker export

#将容器的文件系统导出为tar存档
语法：docker export [OPTIONS] CONTAINER
实例：docker export redis_container > redis_container.tar

# docker port

#列出容器的端口映射
语法：docker port
实例：docker port redis_container

# docker update

#更新一个或多个容器的配置
语法：docker update [OPTIONS] CONTAINER [CONTAINER...]
实例：docker update --cpu-shares 512 redis_container

# docker volume rm

# 容器网络

# docker network create

#创建一个网络
语法：docker network create [OPTIONS] NETWORK
实例：docker network create --driver=bridge --subnet=192.168.0.0/16 redis_network

# docker network connect

#连接容器到某个网络中
语法：docker network connect [OPTIONS] NETWORK CONTAINER
实例：docker network connect redis_network redis_container

# docker network disconnect

#断开容器与某个网络的连接
语法：docker network disconnect [OPTIONS] NETWORK CONTAINER
实例：docker network disconnect redis_network redis_container

# docker network inspect

#显示一个或多个网络的详细信息
语法：docker network inspect [OPTIONS] NETWORK [NETWORK...]
实例：docker network inspect redis_network

# docker network ls

#网络列表
语法：docker network ls [OPTIONS]
实例：docker network ls

# docker network prune

#删除所有未使用的网络
语法：docker network prune [OPTIONS]
实例：docker network prune

# docker network rm

#删除一个或多个网络
语法：docker network rm NETWORK [NETWORK...]
实例：docker network rm redis_network
